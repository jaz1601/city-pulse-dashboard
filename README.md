# City Pulse SA - Dashboard & Skills

Complete dashboard and automation system for managing multi-city newsletters across South Africa.

## 🚀 What's Included

### Dashboard
- **Location:** `/dashboard/`
- **URL:** http://localhost:3000
- **Features:** Article management, drag-and-drop export, city filtering

### Skills
- **Letterman AI** - Newsletter/article management
- **MintBird Manager** - Sales funnels and lead pages
- **City Pulse SA Newsroom** - Multi-city news operations

### Documentation
- **Second Brain** - Complete knowledge base in `/second-brain/`

## 📁 Structure

```
/
├── dashboard/          # Web dashboard
├── skills/            # OpenClaw skills
│   ├── letterman-ai/
│   ├── mintbird-manager/
│   └── city-pulse-sa-newsroom/
├── second-brain/      # Documentation
└── credentials/       # API keys (not in repo)
```
